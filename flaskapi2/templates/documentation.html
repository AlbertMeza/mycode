Introduction:Alberts Pokemon API

This website provides a fun and enjoyable way to make random pokemon teams. We include a neat little photo and some basic information. Who doesn't love pokemon?

What is an API?

An API(Application Programming Interface) is a contract that allows developers to interact with an application through a set of interfaces.

A RESTful API is an API that conforms to a set of loose conventions based on HTTP vers, errors, and hyperlinks.

API Routes

"/"

The default route which randomly assigns you a team of pokemon!

"/get-{1-7}"

If you want a team from a certain region for example the original pokemon(generation 1), we have that option for you

  • Generation one - Kanto
  • Generation two - Johto
  • Generation three - Hoenn
  • Generation four - Sinnoh
  • Generation five - Unova
  • Generation six - Kalo
  • Generation seven - Alola
  • Generation eight - Galar

"/random-pokemon"

This returns a JSON of a randomly assigned pokemon and after you go to the default route, you get a nice photo of that pokemon!

"/pokemon/{name}"

This returns a JSON of a pokemon you want to look up and after you go to the default route, you get a nice photo of that pokemon!
